Laravel 8 with Admin-lte@3.0.1 template

- Authenication
  
## Installation 

Install Larvel8-Adminlte@3.0.1

```bash 
  git clone https://github.com/ndeereedwin/laravel8-Adminlte.git

  composer install

  create database
  cp .env.example .env
  php artisan key:generate
  php artisan migrate
  
  npm install
  npm run dev

```
